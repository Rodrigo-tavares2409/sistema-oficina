# Regras do Projeto (Sistema de Oficina)

<RULE[git_sync]>
- Como este é um repositório compartilhado, o assistente deve sempre:
  1. Rodar `git pull` ANTES de fazer qualquer alteração no código para evitar conflitos com outros desenvolvedores.
  2. Rodar `git add .`, `git commit -m "..."` e `git push` DEPOIS de terminar qualquer tipo de atualização ou implementação para garantir que o código seja compartilhado imediatamente.
</RULE[git_sync]>
